- Lint-Staged
- Vitest
- Cypress
- Vite

## Installation and Setup

1. Clone the project
```
git clone https://github.com/macho-catt/gyul-hap.git
```

2. Install dependencies
```
npm install
```

3. Run locally
```
npm run dev
```
